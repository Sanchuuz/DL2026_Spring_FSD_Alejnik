# Weather Mood (Погода с настроением)

Веб-приложение для мониторинга погоды с функцией подбора контента под текущее состояние атмосферы. Проект реализован в рамках выполнения технического задания для портфолио Fullstack-разработчика.

---

## 🌟 Основные возможности

- **Прогноз на 3 дня:** Интеграция с OpenWeatherMap API для получения точных данных.
- **Интерактивный селектор мемов:** Система автоматически подбирает визуальный контент в зависимости от температуры и осадков.
- **Динамическая локализация:** Поддержка русского и английского языков (i18n).
- **Адаптивная верстка:** Интерфейс оптимизирован для работы на смартфонах и десктопах.

## 🛠 Технологический стек

- **Frontend:** React.js (Hooks, Context API)
- **Backend:** Node.js, Express.js
- **База данных:** MongoDB (хранение пользовательских предпочтений)
- **Внешние API:** OpenWeatherMap API

## ⚙️ Установка и запуск

1.  **Клонирование проекта:**

    ```bash
    git clone [https://github.com/your-username/weather-mood.git](https://github.com/your-username/weather-mood.git)
    cd weather-mood
    ```

2.  **Установка зависимостей:**

    ```bash
    # Для фронтенда
    cd client && npm install
    # Для бэкенда
    cd ../server && npm install
    ```

3.  **Конфигурация (Environment Variables):**
    Создайте файл `.env` в корневой папке сервера:

    ```env
    PORT=5000
    WEATHER_API_KEY=ваш_api_ключ
    MONGO_URI=ваша_строка_подключения_к_mongodb
    ```

4.  **Запуск приложения:**
    ```bash
    # В папке сервера
    npm run dev
    # В папке клиента
    npm start
    ```

## 📂 Структура проекта

- `/client` — Фронтенд на React.
- `/server` — REST API на Node.js/Express.
- `/models` — Схемы данных Mongoose.
